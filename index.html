    Windows 下的 package 管理器: Chocolatey

    Chocolatey (https://chocolatey.org/) 是 React Native 官方推荐的 Windows 环境下的 package 管理器，配合 Power Shell 用起来也挺舒服的，一键安装 Node、Python、JDK、git等，Mark 一下，免得忘记了。
