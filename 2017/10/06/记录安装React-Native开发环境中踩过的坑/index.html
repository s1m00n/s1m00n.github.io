    • 记录安装React Native开发环境中踩过的坑

    最近在安装配置 RN 开发环境时，遇到了一些不大不小的，坑，记录一下，方便以后绕过去：

    1、由于国内的网络环境，如果直接安装官方的安装指南，很容易出错，建议按照某 RN 中文网的教程，把 npm 的镜像源设置为 taobao.org 提供的源，同时安装好yarn。

    2、如果启动过程中出错，很多时候是缺少第三方的库。
